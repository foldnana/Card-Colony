# 《异界旅人》概念图风格 UI 实现说明 v0.3

> 本文记录的是第一版概念 UI。该版本把中心卡牌做成了按钮，遮挡了项目原生的 StackCraft 拖拽桌面，已被 v0.4 的原生堆叠交互方案取代。请以《原生堆叠卡牌交互修正说明 v0.4》为准。

## 1. 本次改造

本次保留 v0.2 已完成的连续时间、行动、卡牌容器和统一存档，彻底替换原先覆盖全屏的功能验证面板，改成与策划案概念图一致的卡牌桌面结构。

玩家现在从世界地图直接操作地点卡，不再从左侧按钮列表选择命令：

```text
点击低语森林地点卡
→ 旅行者卡沿路线移动，时间推进 30 分钟
→ 自动切入低语森林探索地图
→ 点击野生草药资源卡，采集 20 分钟
→ 草药卡进入左下背包卡槽
→ 打开右侧“配方”，点击治疗药水卡
→ 消耗 2 份草药并制作 15 分钟
→ 药水卡进入背包
```

## 2. 界面结构

### 世界地图

- 使用纸面手绘风世界地图作为全屏背景。
- 河湾村、白石城、低语森林、旧矿洞、马车和魔王城均以地点卡显示。
- 旅行者自身也是卡牌，并显示在河湾村附近。
- 开始探索低语森林后，旅行者卡根据行动进度沿蓝色路线移动。
- 探索完成后自动切换到低语森林视图。

### 低语森林

- 使用独立的手绘森林探索底图。
- 森林入口、古老祭坛、林间洞穴、森林狼、草药和木材都是场景卡。
- 当前闭环中，草药资源卡可点击并真正启动采集行动。
- 其余卡牌作为后续调查、战斗和资源系统的视觉插槽。

### 固定 HUD

- 左上：第几天、具体时间、当前状态、暂停、正常、四倍和主动等待。
- 左下：旅行者生命/精力、背包槽位、重量、手持卡和四个可见卡槽。
- 右侧：任务/配方标签页，以及保存、读取按钮。
- 下方：世界地图与低语森林视图切换。
- 中上：当前耗时行动及进度条。

## 3. 卡牌显示

背包卡由 `CardContainer.Cards` 动态实例化，改为竖向卡牌而不是横向数据行。

卡牌显示：

- 物品名称与数量。
- 品质。
- 总重量。
- 批次。
- 取出与拆分操作。

针对项目现有中文 TMP 字体在窄矩形中的溢出行为，地点操作、生命/精力、背包摘要和物品详情均使用明确的多行排版，避免空格后的内容被截断。

## 4. 美术资产

新增背景：

- `Assets/CardColony/Art/Backgrounds/WorldMapBackground.png`
- `Assets/CardColony/Art/Backgrounds/WhisperingForestBackground.png`

两张背景使用内置图像生成工具制作，定位为“无文字、无卡牌、无 HUD 的交互底图”。核心提示要求为：冷灰蓝纸张、铅笔与墨线幻想地图、留出卡牌负空间，并禁止任何文字、按钮、水印或嵌入式 UI。

所有地点卡、资源卡、角色卡、任务、配方和按钮仍由 Unity Prefab 构成，而不是烘焙在背景图片中，因此可以响应点击、改变状态并复用数据。

```text

```

## 6. 当前边界

- 白石城、旧矿洞、马车、魔王城、祭坛、洞穴、森林狼和木材目前是可扩展的场景卡位，尚未接入各自玩法。
- 当前旅行使用行动进度驱动位置插值，还没有曲线路径动画、粒子和音效。
- 资源与物品暂时使用中文字形作为卡面符号，后续可以替换为统一手绘图标。
- 战斗仍保持现有 StackCraft 自动战斗方式，本次没有重写战斗表现。
