# 河湾村地点地图第一阶段实现说明 v0.6

## 本阶段完成内容

- 世界地图右侧地点栏在小队驻扎河湾村时，可点击“进入地点”。
- 进入后加载独立的 `Location` 通用地点场景，当前地点由 `ActiveLocationId` 指定。
- 世界地图上的“小队”汇总卡进入河湾村后，展开为可正常拖拽、堆叠和参与原项目系统的村民人物卡。
- 地点场景底部提供“返回世界地图”按钮。
- 返回时保存地点桌面上的人物卡状态，并恢复世界地图及小队在河湾村地点卡上的驻扎表现。
- 第一阶段只开放河湾村。其它地点仍可作为旅行目的地，但驻扎后按钮显示“地点地图开发中”，不会误入未配置场景。

## 操作流程

1. 新游戏进入世界地图。
2. 点选河湾村地点卡。
3. 小队驻扎河湾村时，点击右侧“进入地点”。
4. 场景切换到河湾村地点桌面，小队汇总卡展开为村民人物卡。
5. 点击底部“返回世界地图”，回到世界地图。

## 技术结构

### 通用地点场景

使用一个可复用的 `Assets/StackCraft/Scenes/Location.unity`，而不是每个地点复制一套 Unity 场景。`LocationDefinition` 负责配置地点 ID、名称、背景和小队展开时使用的人物卡定义。后续增加地点时，只需增加地点定义和地点内容配置。

河湾村目前使用：

- 地点 ID：`riverbend`
- 显示名称：河湾村
- 展开成员卡：`Card_Villager`
- 地点背景：暂用项目已有草地背景，后续替换为正式河湾村桌面美术

### 场景切换和存档

`GameData` 新增：

- `ActiveLocationId`：当前进入的地点 ID
- `PartyMembers`：跨世界地图与地点场景传递的小队成员数据

局部地点存档使用 `Location/{地点ID}` 作为作用域。例如河湾村为 `Location/riverbend`，避免未来多个地点共用 `Location` 场景时互相覆盖桌面卡牌。

进入河湾村前保存世界地图；返回前保存河湾村桌面。再次进入河湾村时，会恢复该地点已经保存的卡牌状态。
